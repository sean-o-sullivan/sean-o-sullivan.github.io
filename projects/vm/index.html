VerifyMe

- Learned Embedding Space Interactive Visualisation

Description

This graph contains 4,493, ≈10K character samples written by 995 authors from The Project Gutenberg Corpus.

Each point represents a text passage that was cleaned, embedded, normalized, processed through EmbeddingNet, and finally all of these reduced from 128 to 3 dimensions using UMAP.
